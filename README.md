ExpenceTracker
The Smart Expense Tracker is a Python project that helps you record daily expenses, categorize them, and analyze spending patterns. It uses a CSV file to store data, validates inputs (date in DD‑MM‑YYYY format, category, amount), and provides monthly summaries with charts. It also includes a logging feature to track actions and errors, plus a log viewer to review past activity.
Execution
Clone or download the project folder.

Ensure all files (main.py, expense_manager.py, data_handler.py, logger.py) are in the same directory.

Install required library:
        pip install matplotlib
Run the program:
        python main.py
